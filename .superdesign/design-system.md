# Triage design system

- Preserve the dark glass-panel dashboard, Pulse Rail, Fraunces headings, Inter UI text, and JetBrains Mono labels.
- Keep the rail distinct from the content viewport; panel peeks must begin to the right of the rail.
- Navigation affordances should be compact, circular, and use existing panel accent colors rather than large full-width bars.
- Use the existing 6px corner radius, subtle light borders, and dark translucent surfaces.
