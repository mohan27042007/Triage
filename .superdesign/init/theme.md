# Theme

The visual system is defined in `frontend/styles.css`: `--ink: #12141A`, `--surface: #1B1E26`, `--paper: #F6F3EC`, red `#E4432C`, amber `#E8A33D`, green `#4F9E7A`, and slate `#5B6470`. Typography uses Fraunces for headings, Inter for UI text, and JetBrains Mono for metadata. Panels are dark translucent glass with a 6–8px radius.
